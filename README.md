## Wallpaper Studio (macOS)
A macOS wallpaper application built with **Flutter**, focused on **design precision** and **UI consistency**.
This project recreates a Figma design to test my ability to follow modern UI principles and produce pixel-perfect layouts.


### Overview
Wallpaper Studio allows users to:

* Browse wallpapers by category
* View and select high-quality wallpapers
* Explore an intuitive and responsive UI that feels native to macOS

The project emphasizes **UI fidelity**, **layout accuracy**, and **component reusability** rather than backend or data-driven complexity.


### Features
* Clean and modern macOS interface
* Category-based wallpaper organization
* Adaptive layouts for desktop scaling
* Smooth transitions and hover effects
* Built with Flutter and ScreenUtil for consistent design metrics

### Tech Stack
* **Framework:** Flutter (macOS desktop)
* **Language:** Dart
* **UI Tools:** Figma (for design reference), Flutter ScreenUtil, Custom Widgets
* **Version Control:** Git
